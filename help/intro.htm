Introducción

NetPortChk es una herramienta de diagnóstico de red para administradores Windows. Verifica la conectividad TCP/UDP hacia puertos específicos de servidores de infraestructura empresarial.

Tipos de servidor soportados

TipoPuertos predeterminados
Domain ControllerDNS 53, Kerberos 88, RPC 135, NetBIOS 139, LDAP 389, SMB 445, LDAPS 636, GC 3268/3269…
Print ServerHTTP 80, HTTPS 443, SMB 445, LPD 515, IPP 631, RAW 9100
SCCM (Full)HTTP/HTTPS 80/443, SMB 445, RPC 135, WSUS 8530/8531
SCCM DPHTTP/HTTPS 80/443, SMB 445, WSUS 8530/8531
Servidor DNSDNS TCP/UDP 53, DNS-over-TLS 853, RNDC 953
Servidor DHCPDHCP Server UDP 67, DHCP Client UDP 68, DHCPv6 546/547
PersonalizadoSin puertos predeterminados.
Requisito: Windows x64. No requiere instalación.
