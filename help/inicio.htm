Primeros pasos

  1. Inicie NetPortChk.exe. Si no existe configuración, se abre el Editor de configuración automáticamente.
  2. Añada los servidores que desea comprobar indicando nombre, IP y tipo.
  3. Revise los puertos de cada servidor y active los que necesite.
  4. Pulse Aceptar para guardar la configuración.
  5. Pulse Comprobar en la barra principal para iniciar el test.
Consejo: La IP de origen se detecta automáticamente y no se guarda en la configuración.

Archivos generados

  • NetPortChk.config — configuración XML.
  • NetPortChk_AAAAMMDD_HHMM.html — informe de resultados.
