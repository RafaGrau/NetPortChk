Informe HTML

Pulse Guardar informe HTML en la barra principal para exportar los resultados.

Cabecera del informe

La cabecera principal muestra dos líneas a 17px:

  • Línea 1: NetPortChk Connectivity Report  —  fecha y hora
  • Línea 2: IP Origen: x.x.x.x      (Timeout: N ms)

Cabecera de servidor

IP Destino: x.x.x.x      Hostname: nombre_servidor  —  (tipo)

Columnas de datos

Puerto, Protocolo, Descripción, Estado, Latencia (ms), Tx (bytes), Rx (bytes).

Solo se muestran los puertos habilitados. El archivo HTML es autocontenido.
