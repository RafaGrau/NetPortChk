Barra de herramientas principal

Los botones se muestran de izquierda a derecha. Los botones Ayuda y Acerca de están fijados al extremo derecho mediante un separador elástico.

Icono Acción Descripción
Comprobar / Detener Inicia o detiene la comprobación de conectividad TCP/UDP.
  Cambiar vista Alterna entre vista lista unificada y vista por pestañas (una pestaña por servidor).
Guardar informe HTML Exporta los resultados a NetPortChk_AAAAMMDD_HHMM.html. Solo activo tras completar una comprobación.
Escaneo RPC dinámico Abre el escáner de puertos RPC (rango 49152–65535). Pre-rellena la IP del servidor seleccionado.
Editor de configuración Abre el editor para gestionar servidores destino y sus puertos.
Guardar configuración Guarda cambios en NetPortChk.config. Solo activo cuando hay cambios sin guardar.
Recargar configuración Recarga NetPortChk.config desde disco descartando cualquier cambio pendiente.
Salir Cierra la aplicación. Si hay cambios sin guardar, pregunta antes de cerrar.
··· Separador elástico Empuja los dos últimos botones al extremo derecho de la barra.
Ayuda Abre NetPortChk.chm. Si el archivo no existe, muestra instrucciones para compilarlo con hhc.exe.
Acerca de Muestra versión, enlace al repositorio GitHub e información sobre los iconos.
Nota: Los botones Guardar configuración y Recargar configuración solo se activan cuando se ha cargado previamente un archivo de configuración válido. Guardar informe HTML solo se activa tras completar al menos una comprobación.
