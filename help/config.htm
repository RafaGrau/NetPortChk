Barra de herramientas del editor

IconoBotónDescripción

⎯ Separador

Borrar puerto Elimina el puerto seleccionado.

Formulario de servidor

  • Nombre — texto identificativo (obligatorio).
  • IP — dirección IPv4 (obligatorio).
  • Tipo — perfil de puertos predeterminados.

Pestañas de servidor

Cada servidor tiene su pestaña. Clic derecho sobre una pestaña para eliminarla.

Lista de puertos

Clic en cabecera Puerto o Protocolo para ordenar ▲/▼.

Menú contextual

  • Clic en fila → Editar puerto / Borrar puerto
  • Clic en área vacía → Agregar puerto

Diálogo de puerto

  • Puerto — número 1–65535 (obligatorio).
  • Protocolo — TCP o UDP.
  • Descripción — se rellena automáticamente para puertos conocidos.
